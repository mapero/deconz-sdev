# deCONZ for ctrlX CORE runtime

## Pre-requirements

Hotplugin is enabled

```bash
sudo snap set system experimental.hotplug=true
```


## Building

`snapcraft`

## Installation

`snap install --dangerous ctrlx-deconz_<version>_amd64.snap`

## Usage

Browse to `http://<ip>:8087`
